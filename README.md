# Welcome to PQRS!

Trabajo final para la materia electiva 1 con el profesor **Jesus David Munera Alvarez**, desarrollado en .net core y .net web API

**Pasos para ejecutar el proyecto**

1. clonar el proyecto: Abrir una consola en la ubicaci√≥n donde se quiere clonar el proyecto y ejecutar el comando 
	    'git clone https://github.com/Mthew2/pqrs.git'
2. en la misma consola navegar a la carpeta contenedora del proyecto con el comando: 'cd pqrs'
3. cambiarse a la rama develop: en la misma consola ejecutar el comando 'git checkout develop'
4. optener los ultimos cambios: en la misma consola ejecutar el comando 'git pull'
5. En el explorador de archivos abrir la carpeta donde quedo clonado el proyecto y abrir la solucion
6. Una vez este abierto el visual studio, abrir el proyecto PQRS.Persistance, y abrir una consola de nuget y ejecutar el siguiente comando 'nuget install packages.config'
